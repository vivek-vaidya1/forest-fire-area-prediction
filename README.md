
# 🔥 Forest Fire Area Prediction Using Machine Learning

## 📌 Project Overview

This machine learning project predicts the burned area of forest fires using environmental and weather-related data.

The project demonstrates data preprocessing, model training, prediction, and evaluation using Python and Neural Networks.

---

## 🛠️ Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* TensorFlow / Keras

---

## 📂 Dataset Features

* Temperature
* Humidity
* Wind
* Rain
* FFMC
* DMC
* DC
* ISI

---

## 🚀 Workflow

1. Import Dataset
2. Data Cleaning
3. Data Preprocessing
4. Train-Test Split
5. Model Building
6. Model Training
7. Prediction
8. Accuracy Evaluation

---

## 📈 Project Outcome

The model predicts the approximate forest fire burned area based on weather and environmental conditions.
